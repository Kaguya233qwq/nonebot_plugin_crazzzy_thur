## nonebot_plugin_kfcrazy
nonebot2肯德基查询插件，输入地区即可查询全国店铺的数据，支持疯狂星期四!
</div>



# 开始使用

克隆本项目到本地，导入包nonebot_plugin_kfcrazy（暂定）

## 使用命令来触发

在有bot的群内发送`/kfc`开始查询肯德基的物品详情
